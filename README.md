# notebookLM_MachineLearning
Este projeto foi desenvolvido a partir de uma proposta de um curso de Inteligência Artificial promovido pela empresa Lupo. O tema escolhido foi Machine Learning, uma das áreas mais relevantes da IA, amplamente aplicada em ciência de dados, análise preditiva e automação.

O objetivo principal deste trabalho é construir uma base sólida sobre os conceitos fundamentais de Machine Learning, seus principais algoritmos e aplicações práticas.
